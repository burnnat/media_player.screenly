# media_player.screenly
Custom component to allow control of [Screenly OSE](https://www.screenly.io/ose/) devices from Home Assistant. 

## Configuration:
```
media_player:
  - platform: screenly
    host: 192.168.1.100
    name: Wall Display
    assets:
      sample: 49c7328e5a24a486539d0f20d1103425
```
